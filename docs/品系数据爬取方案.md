# 滴个龟龟 · 龟类品系数据爬取方案

## 目标

自动化采集蛋龟（牟氏/麝香/剃刀等）品系数据，生成可直接导入数据库的 JSON 条目，替代人工逐条标注。

## 数据源优先级

| 优先级 | 数据源 | 采集内容 | 格式 |
|--------|--------|----------|------|
| P0 | 百度百科 | 品种名、学名、分布、体型、习性、保护等级 | HTML |
| P0 | 维基百科（中文） | 品种名、学名、分类学、分布 | HTML/wiki API |
| P1 | 爬天/灵龟之家论坛 | 品系分类、市场价格、玩家术语 | HTML |
| P1 | iNaturalist | 图片、地理分布 | API |
| P2 | Reptile Database | 分类学、学名对照 | Web |

## 目标数据字段

每个品种/品系输出一条 JSON：

```json
{
  "species_id": "auto",
  "common_name_cn": "剃刀龟",
  "common_name_en": "Razor-backed Musk Turtle",
  "scientific_name": "Sternotherus carinatus",
  "family": "Kinosternidae",
  "genus": "Sternotherus",
  "category": "蛋龟",
  "morph": "原种",
  "morph_tier": "入门",
  "size_max_cm": 15,
  "lifespan_years": 20,
  "distribution": "美国东南部",
  "conservation": "无危(LC)",
  "cites": "否",
  "difficulty": "初级",
  "temp_range": "22-28°C",
  "price_range": "300-800元",
  "key_traits": ["龙骨背甲", "夜行性", "底栖"],

  "source": "百度百科",
  "source_url": "https://baike.baidu.com/item/剃刀龟"
}
```

## 目标品种清单（MVP — 蛋龟）

第一阶段采集以下 10 个品种：

1. 剃刀龟
2. 麝香龟
3. 巨头麝香龟
4. 平背麝香龟
5. 白唇泥龟
6. 红面泥龟
7. 黄泽泥龟
8. 果核泥龟
9. 牟氏水龟
10. 鹰嘴泥龟

## 技术方案

1. **Python 爬虫脚本**
   - `requests` + `BeautifulSoup` 爬百度百科
   - `wikipedia-api` 爬维基百科（兜底）
   - 输出为 `data/species_mvp.json`

2. **字段填充策略**
   - 优先百度百科 → 维基百科兜底
   - 缺失字段标记为 `null`，人工审核时补充
   - `price_range` 从论坛/贴吧抓取（需人工确认）

3. **输出**
   - `scripts/scrape_species.py` — 爬取脚本
   - `data/species_mvp.json` — 输出数据
   - `scripts/import_to_db.js` — 导入 Node.js 数据库脚本

## 给开发智能体的指令

> 写一个 Python 脚本 `scripts/scrape_species.py`，爬取百度百科上 10 个蛋龟品种的数据，输出为 `data/species_mvp.json`。使用 requests + BeautifulSoup，对每个品种：
> 1. 访问 `https://baike.baidu.com/item/{品种名}`
> 2. 解析页面中的学名、分布、体型、习性等字段
> 3. 使用维基百科 API 做兜底
> 4. 将采集到的字段按上述 JSON schema 输出
> 5. 缺失字段标记为 null
> 脚本需支持断点续爬（已爬品种跳过）、请求间隔 3 秒、User-Agent 伪装。
