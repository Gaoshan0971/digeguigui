## 二、汇编作品著作权登记（物种数据库）

### 登记类型
《中华人民共和国著作权法》第14条：汇编作品——"对若干作品、数据或其他材料进行独创性选择或编排形成的汇编作品"

### 独创性说明
- **数据源筛选**: 从全球爬虫学数据库（GBIF、iNaturalist、Reptile Database、IUCN）中独创性筛选宠物爬虫物种
- **中文本地化命名**: 基于中国爬宠圈约定俗成名称，独创性编配拉丁学名→中文名→俗名映射
- **多维信息整合**: 独创性地将分类学(Taxonomy)、分布(distribution)、栖息地(habitat)、保护等级(conservation)、繁殖(reproduction)、词源(etymology)六维整合为统一数据模型
- **品类划分体系**: 独创性地按中国爬宠市场品类划分（龟/蛇/蜥蜴/蛙/守宫五大类，33科）

### 登记材料清单
1. ✅ 数据库设计文档（schema + ER图）
2. ✅ 数据样本（species表前50条记录）
3. ✅ 数据源授权说明（GBIF CC0、iNaturalist CC-BY-NC、ReptileDB非商业引用）
4. ✅ 汇编独创性说明（本文档）
5. ⏳ 作品说明书（需起草）
6. ⏳ 权利归属声明（个人/公司）

### 作品名称建议
《中国爬宠异宠基因库物种分类数据库》  
或《滴个龟龟·异宠百科物种分类数据汇编》

## 三、软件著作权登记

### 3.1 AI识龟引擎软著
- 软件名称：滴个龟龟AI图像识别引擎 V1.0
- 功能：基于混元视觉大模型的龟类品系识别系统
- 代码量：约2000行（API接口+图像预处理+结果映射）
- 提交材料：源代码前后各30页 + 软件说明书

### 3.2 品系基因引擎软著
- 软件名称：滴个龟龟品系基因计算引擎 V1.0
- 功能：爬宠基因遗传计算器，支持隐性/显性/共显/多基因Punnett方阵
- 代码量：约3000行（gene registry + calculator + combo engine）
- 核心模块：
  - genecalc.py（基因计算器）
  - combo_morphs.py（组合品系引擎）
  - morph_genes表（106基因定义）
  - morph_combinations表（73组合公式）

### 软著登记流程
1. 中国版权保护中心官网（www.ccopyright.com.cn）在线申请
2. 提交源代码文档（PDF，前后各30页，约60页）
3. 提交软件说明书
4. 缴费（约300元/件）
5. 受理后约30个工作日取得证书

## 五、竞争壁垒分析

| 壁垒维度 | 滴个龟龟 | 竞品现状 |
|---------|---------|---------|
| 物种覆盖面 | 401→560种（龟366+异宠全量） | 百度百科碎片化/无系统 |
| 数据权威性 | GBIF+iNat+ReptileDB 三源交叉验证 | 多为百度百科/个人博文 |
| 品系基因引擎 | 106基因+73组合+计算器 | 圈内仅靠Excel/手工计算 |
| 中文命名体系 | 中文学名+俗名+圈内称呼 | 命名混乱无标准 |
| 图像授权合规 | CC授权图片+来源署名 | 多为无授权爬图 |
| 数据可引用性 | 每条数据有出处+交叉验证 | 无出处标注 |

**核心壁垒**：国内唯一的合法、权威、结构化、可引用的异宠数据平台。版权登记后将形成法律护城河，禁止未经授权的数据复制和商用。
